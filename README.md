# Puzzle Game

A classic 15-tile sliding puzzle game built with HTML, CSS, and JavaScript.

## Play the Game

🎮 [Click here to play](https://bttaleb.github.io/puzzle-game/)

## How to Play

- Click any tile **adjacent** to the empty slot to slide it into place
- Arrange all tiles in sequential order (1–15) from left to right, top to bottom
- Try to solve the puzzle in as few moves and as little time as possible

## Features

- **New Game** — shuffles the board for a fresh start
- **Simple Game** — sets up a one-move win to test the controls
- Move counter and live timer

## Built With

- HTML
- CSS
- JavaScript
